## STM32代码仓库
### 配置
硬件：正点原子STM32F103ZE精英版 + STLink
### 工具链
Windows:
```c
STM32标准库函数 + mingw-make + STM32CubeProgrammer + arm-gnu-toolchain + Cortex-Debug + STLink-server
```

Linux:
```c
STM32标准库函数 + make + STM32CubeProgrammer + arm-gnu-toolchain + Cortex-Debug + STLink-server
```
